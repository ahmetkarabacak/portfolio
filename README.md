# Kişisel Portfolyo Sitesi

Bu proje, modern ve minimalist bir yaklaşımla tasarlanmış kişisel portfolyo sitesidir.

## Özellikler

- Responsive tasarım
- Modern ve temiz arayüz
- Animasyonlu geçişler (AOS.js kullanılarak)
- Sosyal medya bağlantıları
- Yetenekler ve projeler bölümleri

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- [AOS.js](https://michalsnik.github.io/aos/) - Animasyon kütüphanesi
- [Font Awesome](https://fontawesome.com/) - İkonlar

## Kurulum

1. Projeyi klonlayın:
```bash
git clone [repo-url]
```

2. Proje dizinine gidin:
```bash
cd portfolyo
```

3. `index.html` dosyasını bir web tarayıcısında açın.

## Özelleştirme

- `styles.css` dosyasındaki renk değişkenlerini değiştirerek tema renklerini özelleştirebilirsiniz
- `index.html` dosyasında içeriği kendi bilgilerinizle güncelleyebilirsiniz
- AOS animasyonlarını `data-aos` özelliklerini değiştirerek özelleştirebilirsiniz
